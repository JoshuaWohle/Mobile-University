Mobile-university

An open source Android application to interface with Virtual Learning Environments such as Moodle

Technical

Requirements

  • Download & Install Eclipse Classic (this could be done through other IDEs, but these explanations require eclipse) - http://www.eclipse.org/downloads/packages/eclipse-classic-422/junosr2
  • Download & Install Android development environment - http://developer.android.com/sdk/index.html & http://developer.android.com/sdk/installing/bundle.html

Procedure

  • Import the main project (Mobile University) in eclipse by specifying "Existing Android Code into workspace"
  • Import evernoteLibrary in eclipse by specifying "Existing Android Code into workspace"
  • Import SlidingMenu in eclipse by specifying "Existing Android Code into workspace"
  • rename the newly imported "library" project (the one with the name "library") to "sliderLibrary"
  • make sure both libraries (evernote & slider) have Project -> Properties -> Android -> isLibrary CHECKED
  • Download the android support library & add it to the java build path - http://developer.android.com/tools/extras/support-library.html
  • Switch the android SDK level to Android 4.0 by going to Project -> Properties -> Android and change the Build Target to Google APIs Platform 4.0
  • Add the two Library projects (evernoteLibrary & sliderLibrary) as a library to the Android project by going to Project -> Properties -> Android -> Add Library (for each library).
  • Make sure to remove any previously associated Libraries
  • Make sure both libraries (evernote & slider) have Project -> Properties -> Android -> isLibrary CHECKED
  • Clean up the project by going to Project -> Clean
  • Create an android virtual machine, targetted to Android 2.3 or higher
  • Run the application on the virtual machine or any Android 2.3+ device

Troubleshooting

  • In case you have following error: "Android requires compiler compliance level 5.0 or 6.0. Found '1.7' instead. Please use Android Tools > Fix Project Properties."
  • If you get the error "Could not find evernoteLibrary.apk!" or "Could not ind sliderLibrary.apk!" you should do the following : For both the evernoteLibrary and the sliderLibrary, go to Project -> Clean... and try again. If this does not work, go to Properties -> Android and UNCHECK "is library" for both libraries. If you continue to have trouble, please re-start the said procedure from 0 and make sure to Project -> Clean every time before running.
  • If all else fails, you can also try to right click on your project and select Android tools -> Fix project properties
